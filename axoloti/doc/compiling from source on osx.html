Compiling from source on OSX

Important

Regular users are recommended to download an install a release version, rather than compiling from source. Releases are published at https://github.com/axoloti/axoloti/releases.

Install Java JDK8, recommended Corretto JDK8

Install XCode

Obtain source code from github

Install the GitHub application https://mac.github.com/. Or if you have git installed, open a terminal, and enter

~$> git clone https://github.com/axoloti/axoloti.git
~$> cd axoloti

Compile

still in the terminal...

~$> cd platform_osx
~$> ./build.sh
